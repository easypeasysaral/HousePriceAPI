# House Price Prediction API

This project predicts house prices using a machine learning model and exposes the model through a FastAPI backend.

## Tech Stack
- Python
- FastAPI
- Scikit-learn
- Pandas
- Machine Learning (Regression)

## Features
- Predict house price based on house attributes
- REST API built using FastAPI
- Swagger documentation for testing endpoints

## Run the Project

pip install -r requirements.txt
uvicorn app:app --reload
